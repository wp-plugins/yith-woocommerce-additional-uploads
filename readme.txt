YITH WooCommerce Addional Uploads lets you attach the customized file you need to the orders of your shop.

== Description ==

A concrete way to customize your orders: load a file with your image and complete your order. YITH WooCommerce Addional Uploads offers you an extreme freedom to add something personal to your purchase. A powerful tool that can transform a normal order in the exact product according to your needs.

= Main features =

* General settings of the plugin. You can set a maximum size for the file to send, one or more allowed file extensions and the order status in which you can add a file.
* Allow users to attach a file to the order in the cart page.
* Allow users to attach a file to the order in the checkout page.
* Allow users to attach a file to the order in the order page.

For a more detailed list of options and features of the plugin, please look at the [official documentation](http://yithemes.com/docs-plugins/yith-woocommerce-pdf-invoice/ "Yith WooCommerce PDF Invoice official documentation").

Discover all the features of the plugin and install it in your theme: the result will be extremely satisfying.

== Installation ==

Important: firstly, you have to download and activate the WooCommerce plugin, which is mandatory for YITH WooCommerce Additional Uploads to be working. If the WooCommerce plugin is active, you will just need to activate the YITH WooCommerce Additional Uploads plugin.

= Configuration =

YITH WooCommerce Additional Uploads will add a new tab called "Additional Uploads" inside the YIT Plugins menu item. There you are able to configure all plugin settings.

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH WooCommerce Additional Uploads` from Plugins page.
